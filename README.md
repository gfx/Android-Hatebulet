## Libraries Used

This project is intended to employ modern Android libraries, such as:

* Android Support Libraries
  * CardView
  * Design
  * RecyclerView
* OkHttp
* Retrofit
* RxJava / RxT4A
* Google Dagger
* ThreeTenABP
* Glide

## Build

In your local machine:

```sh
# make an apk and install it to the connected device
./gradlew installDebug
```
